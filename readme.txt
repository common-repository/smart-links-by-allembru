== Description ==
Dynamically link keywords within your posts! Simply enter the keyword(s) you would like to add a link to, enter the URL to be linked to, give your link a title, and finally select whether you would like your link to open in a new window or the current window. Smart Links by Allembru will scan your posts for that word and dynamically insert the assigned link.

== Installation ==
1. Unpack and upload `allembru-slinks.zip` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add your key words and their corresponding links.
